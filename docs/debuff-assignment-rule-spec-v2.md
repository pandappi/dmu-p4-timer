# 디버프 부여 규칙 정리

## 목적

디버프 타이머 서비스에서 사용할 **차수별 디버프 부여 규칙**을 정리한다.

이 문서는 디버프가 어떤 차수에 누구에게 어떤 방식으로 부여되는지, 그리고 1차와 3차 사이의 자동 추론 규칙을 구현하기 위한 기준이다.

---

# 전체 구조

총 8명의 플레이어에게 1차부터 5차까지 디버프가 순차적으로 부여된다.

```text
1차: 물 / 번개 / 가속도폭탄 + 마안
2차: 혼돈의 불 or 혼돈의 물
3차: 물 / 번개 / 가속도폭탄 + 마안
4차: 2차와 반대되는 혼돈 디버프
5차: 알라그필드 or 죽음초월 + 상처 디버프
```

---

# 1차 디버프 부여 규칙

## 부여 구성

1차에는 다음 디버프가 부여된다.

| 디버프   | 영문명               | 인원 |
| ----- | ----------------- | -: |
| 물     | Compressed Water  | 2명 |
| 번개    | Forked Lightning  | 2명 |
| 가속도폭탄 | Acceleration Bomb | 4명 |
| 마안    | Cursed Shriek     | 2명 |

단, **마안은 독립 선택 디버프가 아니라 추가 디버프**로 처리한다.

1차 실제 구조는 다음과 같다.

```text
물 2명
번개 2명
가속도폭탄 2명
가속도폭탄 + 마안 2명
```

즉 1차 마안은 가속도폭탄 대상자 중 2명에게 추가로 붙는다.

---

## 1차 시간 규칙

| 디버프   | 시간              |
| ----- | --------------- |
| 물     | 51초 or 76초      |
| 번개    | 51초 or 76초      |
| 가속도폭탄 | 51초 2명 + 76초 2명 |
| 마안    | 60초 고정          |

1차에서 물과 번개는 항상 같은 시간을 가진다.

```text
1차 물/번개 빠른 패턴: 51초
1차 물/번개 느린 패턴: 76초
```

확인된 규칙:

```text
1차 물 2명은 서로 같은 시간
1차 번개 2명은 서로 같은 시간
1차 물/번개 4명 전체가 같은 시간
```

---

# 2차 디버프 부여 규칙

## 부여 구성

2차는 8명 전원에게 같은 계열의 디버프가 부여된다.

| 디버프   | 영문명           | 인원 |
| ----- | ------------- | -: |
| 혼돈의 불 | Entropy       | 8명 |
| 혼돈의 물 | Dynamic Fluid | 8명 |

둘 중 하나만 나온다.

```text
2차 혼돈의 불
또는
2차 혼돈의 물
```

---

## 2차 시간 규칙

| 디버프   |  시간 |
| ----- | --: |
| 혼돈의 불 | 60초 |
| 혼돈의 물 | 84초 |

---

## 2차와 4차 관계

2차에 나온 디버프에 따라 4차 디버프가 자동으로 정해진다.

```text
2차 혼돈의 불 → 4차 혼돈의 물
2차 혼돈의 물 → 4차 혼돈의 불
```

---

# 3차 디버프 부여 규칙

## 부여 구성

3차에는 1차와 동일한 종류의 디버프가 부여된다.

| 디버프   | 영문명               | 인원 |
| ----- | ----------------- | -: |
| 물     | Compressed Water  | 2명 |
| 번개    | Forked Lightning  | 2명 |
| 가속도폭탄 | Acceleration Bomb | 4명 |
| 마안    | Cursed Shriek     | 2명 |

단, 3차에서도 **마안은 독립 선택 디버프가 아니라 추가 디버프**로 처리한다.

3차 실제 구조는 다음과 같다.

```text
물 2명
번개 2명
가속도폭탄 2명
가속도폭탄 + 마안 2명
```

즉 3차 마안도 가속도폭탄 대상자 중 2명에게 추가로 붙는다.

---

# 1차 ↔ 3차 대상자 전환 규칙

가장 중요한 자동 추론 규칙이다.

## 기본 전환

```text
1차 물/번개 대상자 4명
→ 3차 가속도폭탄 대상자 4명

1차 가속도폭탄 대상자 4명
→ 3차 물/번개 대상자 4명
```

즉 1차에 물 또는 번개가 부여된 사람은 3차에 가속도폭탄이 부여된다.

```text
1차 물 → 3차 가속도폭탄
1차 번개 → 3차 가속도폭탄
```

반대로 1차에 가속도폭탄이 부여된 사람은 3차에 물 또는 번개가 부여된다.

```text
1차 가속도폭탄 → 3차 물 or 번개
```

---

## 마안 포함 규칙

마안은 추가 디버프이므로 기본 전환 규칙과 별개로 처리한다.

```text
1차 물/번개 대상자
→ 3차 가속도폭탄 대상자
→ 그중 일부는 3차 마안도 추가로 받을 수 있음

1차 가속도폭탄 대상자
→ 3차 물/번개 대상자
```

서비스에서는 마안을 기본 디버프 선택지로 두지 않는다.

```text
기본 디버프:
- 물
- 번개
- 가속도폭탄

마안:
- 자동 공통 기믹
- 또는 가속도폭탄 대상자에게 추가되는 보조 디버프
```

---

# 1차 ↔ 3차 시간 보완 규칙

1차 물/번개 시간이 정해지면 3차 물/번개 시간이 자동으로 정해진다.

| 1차 물/번개 시간 | 3차 물/번개 시간 |
| ---------: | ---------: |
|        51초 |        61초 |
|        76초 |        36초 |

즉:

```text
1차 물/번개 51초 → 3차 물/번개 61초
1차 물/번개 76초 → 3차 물/번개 36초
```

이 규칙은 서비스에서 자동 추론에 사용한다.

---

## 3차 시간 규칙

| 디버프   | 시간              |
| ----- | --------------- |
| 물     | 36초 or 61초      |
| 번개    | 36초 or 61초      |
| 가속도폭탄 | 36초 2명 + 61초 2명 |
| 마안    | 69초 고정          |

3차 물과 번개는 항상 같은 시간을 가진다.

```text
3차 물/번개 빠른 패턴: 36초
3차 물/번개 느린 패턴: 61초
```

확인된 규칙:

```text
3차 물 2명은 서로 같은 시간
3차 번개 2명은 서로 같은 시간
3차 물/번개 4명 전체가 같은 시간
```

---

# 4차 디버프 부여 규칙

## 부여 구성

4차도 8명 전원에게 같은 계열의 디버프가 부여된다.

| 디버프   | 영문명           | 인원 |
| ----- | ------------- | -: |
| 혼돈의 불 | Entropy       | 8명 |
| 혼돈의 물 | Dynamic Fluid | 8명 |

4차 디버프는 2차와 반대로 나온다.

```text
2차 혼돈의 불 → 4차 혼돈의 물
2차 혼돈의 물 → 4차 혼돈의 불
```

---

## 4차 시간 규칙

| 디버프   |  시간 |
| ----- | --: |
| 혼돈의 불 | 45초 |
| 혼돈의 물 | 69초 |

---

# 5차 디버프 부여 규칙

## 부여 구성

5차에는 다음 두 계열이 조합되어 부여된다.

## 5차 주요 디버프

| 디버프   | 영문명           |
| ----- | ------------- |
| 알라그필드 | Allagan Field |
| 죽음초월  | Beyond Death  |

둘 중 하나가 부여된다.

## 5차 상처 디버프

| 디버프     | 영문명         | 색상 |
| ------- | ----------- | -- |
| 죽은자의 상처 | Black Wound | 파랑 |
| 산자의 상처  | White Wound | 보라 |

상처 디버프는 색상 판정용이다.

```text
Black Wound = 죽은자의 상처 = 파랑
White Wound = 산자의 상처 = 보라
```

---

## 5차 시간 규칙

| 디버프     |     시간 |
| ------- | -----: |
| 알라그필드   |    15초 |
| 죽음초월    | 14~15초 |
| 죽은자의 상처 |  9999초 |
| 산자의 상처  |  9999초 |

서비스에서는 죽음초월을 기본적으로 15초 처리한다.

---

## 5차 처리 조합

```text
죽음초월이면 상처와 같은 색 맞기
알라그필드면 상처와 다른 색 맞기
```

| 조합                   | 처리    |
| -------------------- | ----- |
| 죽음초월 + 죽은자의 상처 / 파랑  | 파랑 맞기 |
| 죽음초월 + 산자의 상처 / 보라   | 보라 맞기 |
| 알라그필드 + 죽은자의 상처 / 파랑 | 보라 맞기 |
| 알라그필드 + 산자의 상처 / 보라  | 파랑 맞기 |

---

# 서비스 입력 최적화 규칙

## 1차 입력

1차에서는 다음만 입력받는다.

```text
- 진실 / 거짓
- 물/번개 시간: 51초 or 76초
```

마안은 직접 선택하지 않는다.

---

## 2차 입력

2차에서는 다음을 입력받는다.

```text
- 진실 / 거짓
- 혼돈의 불 or 혼돈의 물
```

---

## 3차 입력

3차에서는 다음만 입력받는다.

```text
- 진실 / 거짓
```

3차 물/번개 시간은 1차 물/번개 시간으로 자동 추론한다.

```text
1차 51초 → 3차 61초
1차 76초 → 3차 36초
```

마안은 직접 선택하지 않는다.

---

## 4차 입력

4차에서는 다음만 입력받는다.

```text
- 진실 / 거짓
```

4차 디버프 종류는 2차 디버프 종류로 자동 추론한다.

```text
2차 혼돈의 불 → 4차 혼돈의 물
2차 혼돈의 물 → 4차 혼돈의 불
```

---

## 5차 입력

5차에서는 다음을 입력받는다.

```text
- 알라그필드 or 죽음초월
- 죽은자의 상처 / 파랑 or 산자의 상처 / 보라
```

5차 입력 완료 시점부터 전체 타이머를 시작한다.

```ts
} as const;
```

# 핵심 결론

```text
1차 물/번개 대상자는 3차 가속도폭탄 대상자다.
1차 가속도폭탄 대상자는 3차 물/번개 대상자다.

1차 물/번개 시간이 51초면 3차 물/번개는 61초다.
1차 물/번개 시간이 76초면 3차 물/번개는 36초다.

2차 혼돈의 불이면 4차 혼돈의 물이다.
2차 혼돈의 물이면 4차 혼돈의 불이다.

마안은 직접 선택하지 않고, 1차/3차 공통 기믹으로 자동 처리한다.
5차 입력 완료 시점부터 전체 타이머를 시작한다.
```
